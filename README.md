# TIC-TAC-TOE
# 🎮 Tic Tac Toe (GUI using Tkinter)

## 📌 Description
This is a simple **Tic Tac Toe game** built using **Python and Tkinter GUI library**.  
It allows two players to play the classic game on a 3×3 grid with an interactive graphical interface.

---

## 🚀 Features
- 🧑‍🤝‍🧑 Two-player game (Player X and Player O)
- 🎨 GUI-based interface using Tkinter
- 🔄 Reset button to restart the game
- 🏆 Displays winner
- 🎯 Highlights player moves with colors
- 🚫 Disables buttons after game ends

---

## 🛠️ Technologies Used
- Python 🐍
- Tkinter (GUI Library)

---

## ▶️ How to Run

1. Make sure Python is installed on your system  
2. Clone this repository:
```bash
git clone https://github.com/your-username/tic-tac-toe-gui.git
```

3. Navigate to the project folder:
```bash
cd tic-tac-toe-gui
```

4. Run the program:
```bash
python filename.py
```

---

## 🎮 How to Play
- Player **X** starts the game
- Players take turns clicking on empty boxes
- First player to align **3 symbols (row, column, or diagonal)** wins
- Click **Reset** to start a new game
